## 스킨 편집

jinja2 템플릿엔진 이용.





## 스킨 적용

### 템플릿 렌더링

`go.py`을 실행하면 산출물이 나온다.

```
python go.py
```

ModuleNotFoundError: No module named 뭐뭐 발생시 requirements 설치.

```
pip install -r requirements.txt
```



### 티스토리

산출물을 티스토리 블로그 스킨 등록.
